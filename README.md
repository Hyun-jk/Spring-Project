## 프로젝트 소개

* 프로젝트명: Premiere (영화 예매사이트)
* 개발 기간: 2022.03.14~2022.04.11
* 인원 : 5명
* 개발 배경
  - 사람들이 많이 애용하는 문화생활 중 하나 인 영화를 주제로 선정
  - Spring Framework를 보다 쉽게 적응하기 위하여 MVC패턴이 두드러진 주제를 선정함
